## Backend Setup

### Prerequisites
- [Node.js](https://nodejs.org/) (version 12+)
- [npm](https://www.npmjs.com/)

### Installation

1. Navigate to the `backend` folder:

   ```bash
   cd backend
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the server:

   ```bash
   npm run dev
   ```

   The server will run on `http://localhost:5000`.
